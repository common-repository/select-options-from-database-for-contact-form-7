Extend contact form 7 with select options from database tag. Any column in your database can be a source to this select options element.

== Description ==

This plugin provides new tag type for the Contact Form 7 Plugin : Select options from database.
You can select any field from your database and let the visitors choose value for you.
You can use it to send each contact form to any user in your site.
You can let the customer contact you according some special post, page, comment or product.
You can chose which value to show and which to send via email.

[Read More](http://dynamicplugin.com/select-options-field-in-wordpress-contact-form-7/ "Read More")

= How to use it =

Select button with option "Select options from database", choose table, column to show, and column to value.
You can add field tag to emails in any field, even as an email option.

== Installation ==

This section describes how to install the plugin and get it working.

1. Download and install the Plugin
1. Upload the plugin folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. You will now have a "Select options database field" tag option in the Contact Form 7 tag generator
